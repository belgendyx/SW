External APIs (You Implement)

External entities must implement these endpoints for SW Platform to call:

POST
Request Submission API
Submit approval request data including request info, applicant details, company information, partners, managers, branches, and attachments.
/submission
POST
Request Completion API
Complete approval requests with payment receipt details after payment is processed.
/completion
POST
Cancel Request API
Cancel an existing approval request using the approvalRequestId.
/api/v1/approval-requests/{id}/cancel
POST
Manual Approval API
Submit manual approval data with approval number, validity dates, and optional attachments.
/manual-approval

SW Provided APIs (We Provide)

SW Platform provides these endpoints for external entities to call:

POST
Approval Decisions Callback
Send approval decision updates (APPROVED, REJECTED, RETURNED) to SW Platform.
/api/v1/approval-decisions
