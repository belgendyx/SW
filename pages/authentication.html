Authentication Overview

Entities may implement authentication using either Bearer JWT or Basic Auth depending on their internal system capabilities.

All API requests require authentication headers to ensure secure communication.

JWT Bearer Authentication

Request Header

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Token Structure

JWT tokens should include:

  • Entity identifier
  • Token expiration time
  • Appropriate claims for authorization

Basic Authentication

Request Header

Authorization: Basic dXNlcm5hbWU6cGFzc3dvcmQ=

Basic Auth header is Base64-encoded username:password

Security Considerations

  • Always use HTTPS for all API calls
  • Rotate credentials regularly
  • Store credentials securely
  • Never commit credentials to source control
