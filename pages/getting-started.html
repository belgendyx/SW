Getting Started

Quick start guide to integrate with SW Platform in under 2 weeks.

Steps
  • 1. Register
  • 2. Download Specs
  • 3. Implement APIs
  • 4. Test
  • 5. Go Live

Step 1: Register for Integration

Contact SW Platform team to register for API access:

  • Provide organization details
  • Specify technical contact
  • Describe integration use case

Timeline: Credentials provided within 24-48 hours

Step 2: Download OpenAPI Specifications

Download all required YAML files:

Submission API Completion API Cancel API

Step 3: Implement Required APIs

Implement 5 REST API endpoints:

  1. POST /submission - Receive approval requests
  2. POST /completion - Receive completion data
  3. POST /cancel - Handle cancellations
  4. POST /manual-approval - Process manual approvals
  5. POST to SW callback - Send approval decisions

Step 4: Test Your Integration

Use our Interactive API Explorer to:

  • Test request/response formats
  • Validate payload structures
  • Test error handling
  • Verify authentication

Step 5: Go Live

Once testing is complete:

  1. Notify SW Platform team
  2. Receive production credentials
  3. Switch to production endpoints
  4. Monitor initial transactions
  5. You're live! 🎉
