Introduction

This document defines the formal integration standards for external entities exposing services to the SW platform. The SW platform consumes external entity APIs using standardized payload structures and shared OpenAPI (Swagger) specifications.

This guide describes integration architecture, responsibilities, security requirements, and operational expectations. Detailed endpoint schemas are defined in the attached Swagger specifications.

Available API Services

External entities implement these APIs for SW platform integration:

POST
Request Submission API
Submit approval request data including request info, applicant details, company information, and approval forms.
POST
Request Completion API
Complete approval requests with payment details and receipt information.
POST
Cancel Request API
Cancel an existing approval request using the approvalRequestId.
POST
Manual Approval API
Submit manual approval data with approval number, validity dates, and attachments.
POST
Approval Decisions Callback
SW-provided callback API for approval decision updates (APPROVED/REJECTED/RETURNED).
